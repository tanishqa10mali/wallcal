WallCal

WallCal is an interactive wall calendar web application built using Next.js. The aim of this project is to provide a clean and user-friendly interface for managing dates, adding notes, and viewing calendar information.

Features:
  Select single dates or a range of dates using drag
  Add sticky notes for reminders
  Light, Dark, and Sepia themes
  Smooth month transition animations
  Indian holiday highlights
  Keyboard navigation support
  Export the current calendar view as an image
Tech Stack:
  Next.js 14
  TypeScript
  Tailwind CSS
  Framer Motion
  date-fns
  Zustand
Project Structure:
  app/ → Main pages and layout
  components/ → Reusable UI components
  styles/ → Global styles
  
How to Run Locally

Clone the repository:

git clone https://github.com/YOUR_USERNAME/wallcal.git
cd wallcal

Install dependencies:

npm install

Run the development server:

npm run dev

Open in browser:

http://localhost:3000
Notes
The project mainly focuses on frontend design and interactivity
Zustand is used for simple and efficient state management
Framer Motion is used to improve user experience with animations
